So what is programming?

All the things that you see on your smartphones, computers, smartwatch, telivions are created by programming. The way we do programming, is by talking to the machine and for talking we obviously need a language.... You got it. That's why we have programming languages. Programming languages differs by what kind of program we have to write for example if we want to create a website we generally use something called html and javascript, or if we want to create a app for your smartphone we generally use java(for android) or swift(for iphones). So basically you can say programming is different based on the environment. Read more


What are programming languages?

Programming languages are of course language but of special kind. And each programming language has its own speciality. Some are fast, some are slow some are functional some are object oriented(don't get confused, this is just a style), some are loved, some are hated.
